# Auth System

This is a simple authentication system built with **Node.js**, **Express**, **MongoDB**, and **Passport.js**. It allows users to register, log in, and log out using securely stored user credentials.

## Features

- **User Registration**: Users can register by providing a username and password.
- **Login**: Users can log in with their credentials and access protected content.
- **Logout**: Users can log out securely.
- **Security**: Passwords are hashed using **bcryptjs** before being stored in the database.
- **Flash messages**: Success and error messages for user feedback during the authentication process.

## Technologies Used

- **Node.js**: JavaScript runtime environment.
- **Express**: Web server framework for Node.js.
- **MongoDB**: NoSQL database to store user information.
- **Passport.js**: Authentication middleware for Node.js.
- **bcryptjs**: Library for securely hashing and comparing passwords.
- **connect-mongo**: Session storage in MongoDB.
- **EJS**: Templating engine to render HTML views.
